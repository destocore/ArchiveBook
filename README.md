# 📚 ArchiveBook — Цифровой архив конспектов

**ArchiveBook** — это веб-сервис на базе Django, созданный для удобного хранения, поиска и мгновенного просмотра учебных материалов. Больше никакого хаоса в чатах — все лекции структурированы и доступны в один клик.

# 🚀 Основные фишки

*   **Умный поиск:** Если запрос совпадает с названием файла, система сразу открывает его в браузере.
*   **Динамическая структура:** Бэкенд сканирует папки в реальном времени. Добавили файл на диск — он тут же появился на сайте.
*   **Безопасность:** Реализована защита медиа-путей (`path normalization`) и разграничение прав (админ видит скрытые папки).
*   **Автоматизация:** При удалении записи из админки файл физически удаляется с сервера (чистота данных).
*   **Минималистичный дизайн:** Чистый интерфейс с удобными кнопками-«таблетками».

# 📂 Структура архива

Файлы распределены по категориям в папке `media/notes/`

Чтобы добавить раздел - нужно создать папку с названием предмета

# Доступные URL страницы

Главная страница:
```bush
/
```

Админ панель:
```bush
/admin
```

Система поиска:
```bush
/find/*категория*/*тема*
```

Система открытия файла:
```bush
/media/notes/*категория*/*тема*
```

# 🔧 Как запустить локально

## Windows

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/destocore/ArchiveBook.git
   ```
2. Установите зависимости (Django):
   ```bash
   pip install -r pip.txt
   ```
3. Запустите сервер:
   ```bash
   python manage.py runserver
   ```
4. Откройте в браузере: `http://127.0.0.1:8000`

## Linux / MacOS / BSD

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/destocore/ArchiveBook.git
   ```
2. Установите зависимости (Django):
   ```bash
   pip3 install -r pip.txt
   ```
3. Запустите сервер:
   ```bash
   python3 manage.py runserver
   ```
4. Откройте в браузере: `http://127.0.0.1:8000`

## Docker

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/destocore/ArchiveBook.git
   ```
2. Сборка контейнера:
    ```bush
    docker build -t archivebook .
    ```
3. Запуск контейнера:
    ```bush
    docker run -p 80:80 archivebook
    ```
4. Откройте в браузере: `http://127.0.0.1:80`
